
<div class="col-sm-6 justo">
    
<p>Se define a la transformada de Fourier de una función \(f(t)\) como</p>
    
<div id="ecuacion" class="col-sm-offset-1 col-sm-10 col-sm-offset-1">
            \begin{equation}
  \mathscr{F}\left\{ f(t)  \right\}=\int _{ -\infty  }^{ \infty  }{ f(t) } { e }^{ -j\omega t }dt
  \label{defFou}\end{equation}
</div>
        
   <p class="justo">siempre y cuando esta integral converja.</p>
    
     <div class="video-ww">
<div class="video-container">
         <iframe src="https://www.youtube.com/embed/dIkGVtuKUmE?start=22&rel=0&showinfo=0&theme=light" allowfullscreen frameborder="0" width="100%" height="250"></iframe>
</div>
</div>

    <p class="justo">Obsérvese que en esta integral desaparece la letra de integración (cuando se termina de integrar), en este caso la \(t\), por lo que este procedimiento da lugar a una nueva expresión en términos de una nueva variable \(\omega\), y dependiendo de la literatura que se consulte se define la siguiente notación:</p>


<div id="ecuacion" class="col-sm-offset-1 col-sm-10 col-sm-offset-1">
\begin{equation}
\mathscr{F}\left\{ f(t)  \right\}=F(j\omega)=F(\omega)
\end{equation}
    </div>
    
<p class="justo">

Se utilizará cualquiera de estas notaciones de manera indistinta. La transformada de Fourier es un tipo de operación que en general se conoce como transformada integral</p>

      <p class="justo">
       <strong>Ejemplo 1.</strong><br>

            Encuentre la transformada de Fourier de \(f(t)={ e }^{ -at }u(t)\)
</p>



<div id="fou1" class="overlay">
  <a href="javascript:void(0)" class="closebtn" onclick="Fou1c()">×</a>
  <div class="overlay-content">
   
  
<p >
        
          Por definición, </p>


          \begin{align*}
          \mathscr{F}\left\{ f(t) \right\}&=	F(\omega )\\
          &=\int _{ -\infty  }^{ \infty  }{ { e }^{ -at } }
            u(t){ e }^{ -j\omega t }dt\\
            &=\int _{ 0 }^{ \infty  }{ { e }^{ -(a+j\omega )t } } dt\\
            &=\frac{ -1 }{ a+j\omega } \left( { \left. { e }^{ -(a+j\omega )t } \right|  }_{ 0 }^{ \infty  }\right)
          \end{align*}


        <p >  Si \( \mathbb{R}e(a)>0\) la evaluación
          \({ \left. { e }^{ -(a+j\omega )t } \right|  }_{ 0 }^{ \infty  }=1\). Por lo tanto:
</p>

          \begin{equation}
            F(\omega )=\frac { 1 }{ a+j\omega  }  \quad \mathbb{R}e(a)>0
          \end{equation}
